# Clawy Notes - Team Directory

## Project: Clawy Notes
**Type:** Markdown Note Editor (Offline-First, Mobile-Ready)
**Stack:** React 18 + TypeScript + Tailwind CSS v3 + Vite + Zustand
**Design:** Editorial/Magazine - Strong grid, dramatic headlines, whitespace as feature
**Reference:** Notion (simplicity) + Bear (beauty) + Obsidian (power)

## Communication Protocol

### Daily Standups (Async)
- Each squad posts status in `#standup-[squad]`
- Format: Yesterday / Today / Blockers

### Sprint Planning
- 2-week sprints
- Monday 9:00 UTC - Sprint Planning
- Friday 16:00 UTC - Sprint Review

### Code Review
- All PRs need 2 approvals
- Design Squad reviews UI changes
- Core Squad reviews state/logic changes

### Deployment
- Staging: Auto-deploy on PR merge to `develop`
- Production: Manual deploy from `main` on Friday
